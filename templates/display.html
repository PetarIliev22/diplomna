ParQly | ANPR Parking System

  • Intelligent ANPR application for automatic vehicle license plate recognition in parking environments.
  • Displays detected license plates and access status in real time.
  • Simulates a smart parking totem used at parking entrances and exits.
  • Integrated with an Automatic Number Plate Recognition system (YOLO + OCR / CRNN).
  • Automatically refreshes every 0.5 seconds to check for newly detected plate data.
  • Developed as part of the ParQly project.
  • Designed for real-world smart parking and access control systems.
